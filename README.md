# AI 学习路线图 2026

> 从零基础到专业方向的系统化学习路径

👉 **在线查看**: [ai-learning-roadmap](https://dray1220.github.io/ai-learning-roadmap/)

---

## 概览

一个干净、可交互的 AI 学习路线图，涵盖四个阶段、18 项核心技能、14 个推荐资源和 15 个实战项目。

- **中英双语** — 一键切换中文 / English
- **明暗主题** — 适配系统偏好
- **进度追踪** — 点击阶段圆点标记完成状态，localStorage 持久化
- **响应式** — 手机、平板、桌面端自适应

## 四阶段

| 阶段 | 内容 | 预计时间 |
|------|------|----------|
| 基础阶段 | Python 编程、线性代数、概率统计、微积分 | 2–3 个月 |
| 机器学习 | 经典算法、数据处理、特征工程、模型评估 | 3–4 个月 |
| 深度学习 | 神经网络、CNN、RNN/LSTM、Transformer、PyTorch | 4–6 个月 |
| 专业方向 | NLP、CV、强化学习、LLM、MLOps | 6+ 个月 |

## 每个阶段包含

- **核心技能卡片** — 技术栈与关键概念
- **前置要求** — 进入该阶段前需掌握的内容
- **推荐资源** — 课程、书籍（附链接）
- **实战项目** — 可写进简历的练手项目

## 技术栈

- 纯静态 HTML/CSS/JS，零依赖，零构建
- 单文件部署，GitHub Pages 托管
- 客户端 i18n（`data-i18n` + JS 翻译对象）
- 设计规范遵循 [DESIGN.md](DESIGN.md)（Google 开放设计令牌规范）
