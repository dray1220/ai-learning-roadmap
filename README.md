# AI 学习路线图 2026

系统化 AI 学习路径，从零基础到专业方向。

🌐 [在线查看](https://dray1220.github.io/ai-learning-roadmap/)

## 四个阶段

1. **基础阶段** (2-3月) — Python、线性代数、概率论、微积分
2. **机器学习** (3-4月) — 经典算法、数据处理、特征工程、模型评估
3. **深度学习** (4-6月) — 神经网络、CNN、RNN/LSTM、Transformer、PyTorch
4. **专业方向** (6月+) — NLP/LLM、计算机视觉、强化学习、MLOps

每个阶段包含推荐课程、实战项目和学习资源。
